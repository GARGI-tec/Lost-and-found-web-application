# Lost and Found Web Application

A campus web app that helps students, faculty, and staff report and recover lost items.

## Features
- Post lost or found items with name, description, location and date
- Upload multiple images of items
- Search and filter listings
- Admin dashboard to manage and approve matches
- Auto fuzzy matching of lost and found items
- Email notifications to users
- PDF report generation

## Tech Stack
- Backend: Python, Flask
- Database: MySQL
- Image Storage: Cloudinary
- Email: Flask-Mail
- PDF: ReportLab
